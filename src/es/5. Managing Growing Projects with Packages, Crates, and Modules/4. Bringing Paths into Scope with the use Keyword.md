# Trayendo Rutas al Ámbito con la Palabra Clave use
La palabra clave **use** trae una ruta al ámbito actual, permitiéndote referirte al elemento por su nombre sin tener que escribir la ruta completa cada vez.

### Ejemplo
El siguiente código demuestra el uso de `use` para simplificar la ruta hacia `Asparagus`.

    // En src/main.rs
    use my_project::garden::vegetables::Asparagus;
    
    fn main() {
        let plant = Asparagus {
            name: String::from("Spargel"),
        };
    }

Puedes usar rutas anidadas con llaves `{}` para traer múltiples elementos de la misma ruta al ámbito,
**como `use std::cmp::{self, Ordering};`**.

También puedes usar la palabra clave **as** para crear un alias, lo cual es útil para evitar conflictos de nombres.
Por ejemplo: **`use std::fmt::Result as FmtResult;`**
