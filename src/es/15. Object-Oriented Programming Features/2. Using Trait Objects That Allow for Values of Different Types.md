# Trait Objects

`Box<dyn Trait>` permite **despacho dinámico** — llamar al método correcto en tiempo de ejecución.

```rust
trait Draw { fn draw(&self); }

struct Button { width: u32 }
impl Draw for Button {
    fn draw(&self) { println!("button {})", self.width); }
}

struct SelectBox { options: Vec<String> }
impl Draw for SelectBox {
    fn draw(&self) { println!("select box with {} options", self.options.len()); }
}

// Collection of different types that all implement Draw
let components: Vec<Box<dyn Draw>> = vec![
    Box::new(Button { width: 50 }),
    Box::new(SelectBox { options: vec!["A".into(), "B".into()] }),
];

for c in components { c.draw(); }
```

- Utiliza una **vtable** (tabla de despacho virtual) en tiempo de ejecución.
- Pequeño costo en tiempo de ejecución comparado con el despacho estático genérico.

> **Punto clave:** Usa trait objects cuando necesites una colección de **tipos diferentes** que compartan un trait.
