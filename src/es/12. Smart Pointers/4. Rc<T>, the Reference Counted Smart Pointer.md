# `Rc<T>` — Conteo de Referencias

Propiedad múltiple en código **monohilo**.

```rust
use std::rc::Rc;

let a = Rc::new(String::from("shared"));
println!("{}", Rc::strong_count(&a));    // 1

let b = Rc::clone(&a);                   // barato: incrementa el contador
println!("{}", Rc::strong_count(&a));    // 2
```

- Los datos se liberan cuando el contador llega a 0.
- `Rc::clone` es barato — solo incrementa un contador.

> **Advertencia:** Usa `Arc<T>` (atómico) para código multihilo. `Rc<T>` no es `Send`.
