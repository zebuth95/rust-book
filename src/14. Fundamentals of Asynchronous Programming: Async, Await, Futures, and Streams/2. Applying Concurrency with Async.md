# Applying Concurrency with Async
Asynchronous concurrency is achieved by having an executor like **Tokio runtime** concurrently poll multiple independent
futures on a limited number of OS threads. This is not parallelism; it's **interleaving** of tasks.

The async block is the primary way to define a block of code as a Future. Using await allows cooperative multitasking.


    use tokio::time::sleep;
    use std::time::Duration;

    #[tokio::main]
    async fn main() {
        start_work().await;
    }
    
    // Imagine this is running on an executor like Tokio
    async fn start_work() {
        let f1 = async {
        // This task starts
        println!("Task 1: Started");
        sleep(Duration::from_millis(100)).await;
        println!("Task 1: Finished");
        };
    
        let f2 = async {
            // This task starts almost simultaneously, before f1 finishes
            println!("Task 2: Started");
            sleep(Duration::from_millis(50)).await;
            println!("Task 2: Finished");
        };
    
        // Tasks run concurrently, interleaved by the executor.
        // tokio::join! executes the futures concurrently and waits for all of them.
        tokio::join!(f1, f2);
    }
