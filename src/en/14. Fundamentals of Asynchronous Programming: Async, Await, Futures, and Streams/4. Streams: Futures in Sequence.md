# Streams: Futures in Sequence

A **Stream** is an **async iterator** — values arrive one at a time over time (network packets, DB rows, sensor readings).

| Trait | Method | Output |
|-------|--------|--------|
| `Iterator` | `.next()` | `Option<Item>` |
| `Stream` | `.next().await` | `Poll<Option<Item>>` |

## Basic Stream Usage

```rust
use tokio_stream::StreamExt;

async fn process(stream: impl Stream<Item = i32>) {
    tokio::pin!(stream);
    while let Some(item) = stream.next().await {
        println!("Got: {item}");
    }
}
```

## Creating a Stream

```rust
use tokio::time::{sleep, Duration};
use tokio_stream::StreamExt;
use futures::stream;

#[tokio::main]
async fn main() {
    // Create a stream that yields numbers 1..=5 with 100ms delay each
    let s = stream::iter(1..=5).then(|n| async move {
        sleep(Duration::from_millis(100)).await;
        n
    });

    tokio::pin!(s);
    while let Some(n) = s.next().await {
        println!("Received: {n}"); // Each item arrives ~100ms apart
    }
}
```

## Stream Combinators

```rust
use tokio_stream::StreamExt;

let s = stream::iter(1..=10);

s.map(|n| n * 2)           // transform each item
 .filter(|n| *n > 5)       // keep only items matching a predicate
 .take(3)                   // take only first 3 items
 .collect::<Vec<_>>().await // collect all into a collection
```

## Key Combinators

| Method | What it does |
|--------|-------------|
| `.next().await` | Get the next item (awaits readiness) |
| `.map(f)` | Transform each item |
| `.filter(p)` | Keep items matching a predicate |
| `.take(n)` | Take first `n` items |
| `.collect()` | Collect into a collection |
| `.then(f)` | Apply an async function to each item |

---

## Key Points

| Concept | Meaning |
|---------|---------|
| **Stream** | Like `Iterator` but each `.next()` may need to await. |
| **`tokio_stream::StreamExt`** | Provides `.next()` and other combinators on streams. |
| **`tokio::pin!`** | Pins the stream in memory (needed because `Stream` includes self-references). |
| **Lazy** | Streams don't do work until polled. |
