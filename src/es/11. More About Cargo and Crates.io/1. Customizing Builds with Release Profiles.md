# Perfiles de Compilación

Dos perfiles predeterminados:

| Perfil | Comando | Optimización | Uso |
|---|---|---|---|
| `dev` | `cargo build` | Ninguna (`opt-level = 0`) | Compilación rápida |
| `release` | `cargo build --release` | Máxima (`opt-level = 3`) | Producción |

Personaliza en `Cargo.toml`:

```toml
[profile.dev]
opt-level = 0

[profile.release]
opt-level = 3
```

> **Nota:** `cargo build --release` compila más lento pero produce código más rápido.
