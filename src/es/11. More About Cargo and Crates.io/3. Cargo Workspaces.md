# Espacios de Trabajo (Workspaces)

Un **workspace** gestiona múltiples crates en un solo proyecto. Un único `Cargo.lock`, salida compartida.

```
my_workspace/
├── Cargo.toml          # [workspace] members = ["adder", "add_one"]
├── adder/
│   └── src/main.rs
└── add_one/
    └── src/lib.rs
```

```toml
# Cargo.toml raíz
[workspace]
members = ["adder", "add_one"]
```

```bash
cargo build   # construye todos los miembros del workspace
```

> **Nota:** Usa workspaces para proyectos grandes compuestos por múltiples crates.
