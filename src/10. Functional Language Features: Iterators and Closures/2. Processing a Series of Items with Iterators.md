# Iterators

Iterators are **lazy** — they do nothing until consumed.

```rust
let v1 = vec![1, 2, 3];

let iter = v1.iter();            // lazy, nothing happens
let sum: i32 = iter.sum();       // consuming adapter, drives iteration
```

## Common adapters

```rust
let v = vec![1, 2, 3];

let mapped: Vec<_> = v.iter().map(|x| x + 1).collect();
assert_eq!(mapped, vec![2, 3, 4]);

let evens: Vec<_> = v.iter().filter(|&&x| x % 2 == 0).collect();
assert_eq!(evens, vec![2]);
```

## `next()` method

```rust
let mut iter = vec![1, 2].iter();
assert_eq!(iter.next(), Some(&1));
assert_eq!(iter.next(), Some(&2));
assert_eq!(iter.next(), None);
```

> **Key point:** Iterators are zero-cost — they optimize to the same code as hand-written loops.
