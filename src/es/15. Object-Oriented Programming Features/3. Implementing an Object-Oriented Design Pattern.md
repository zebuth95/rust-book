# Patrón State con Trait Objects

Un patrón POO común — cambia el comportamiento intercambiando el estado interno.

```rust
trait State {
    fn request_review(self: Box<Self>) -> Box<dyn State>;
    fn approve(self: Box<Self>) -> Box<dyn State>;
    fn content<'a>(&self, post: &'a Post) -> &'a str { "" }
}

struct Draft {}
impl State for Draft { /* ... */ }

struct Published {}
impl State for Published {
    fn content<'a>(&self, post: &'a Post) -> &'a str { &post.content }
}

pub struct Post {
    state: Option<Box<dyn State>>,
    content: String,
}
```

> **Nota:** Cada estado es un tipo concreto que implementa el trait `State`. El `Post` delega el comportamiento a su estado actual.
