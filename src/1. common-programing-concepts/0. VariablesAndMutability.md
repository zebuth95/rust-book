# Variables and Mutability

Variables are **immutable** by default.

## `let` (immutable)

```rust
let x = 5;
// x = 6;  // ❌ compile error: cannot assign twice
```

## `let mut` (mutable)

```rust
let mut x = 5;
x = 6;  // ✅ allowed
```

## `const` (always immutable)

```rust
const MAX_POINTS: u32 = 100_000;
```

## Shadowing

Re-declare with `let` to create a new variable (can change type).

```rust
let x = 5;
let x = x + 1;       // shadows, x = 6
let x = "hello";     // shadows with different type
```

> **Key point:** Shadowing is NOT mutation. The old value is replaced.
