# Usando Structs — Ejemplo

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

fn main() {
    let rect = Rectangle { width: 30, height: 50 };
    println!("Area: {}", area(&rect));
}

fn area(r: &Rectangle) -> u32 {
    r.width * r.height
}
```

> **Nota:** Toma prestado el struct (`&Rectangle`) para evitar mover la propiedad.
