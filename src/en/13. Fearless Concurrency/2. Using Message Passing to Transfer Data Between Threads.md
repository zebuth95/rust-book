# Message Passing (Channels)

Use `mpsc` (Multiple Producer, Single Consumer) channels.

```rust
use std::sync::mpsc;
use std::thread;

let (tx, rx) = mpsc::channel();

thread::spawn(move || {
    tx.send(String::from("hello")).unwrap();
});

let received = rx.recv().unwrap();    // blocking
println!("{received}");
```

- `send()` moves ownership to the channel.
- `recv()` blocks until data arrives.
- `try_recv()` returns immediately.

> **Key point:** Ownership rules prevent races — once sent, sender can't use the value.
