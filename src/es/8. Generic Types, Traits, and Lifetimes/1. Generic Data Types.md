# Tipos de Datos Genéricos

Escribe funciones, estructuras, enumeraciones y métodos que funcionen con **cualquier tipo** — sin duplicación de código.

## Función Genérica

```rust
fn largest<T: std::cmp::PartialOrd>(list: &[T]) -> &T {
    let mut largest = &list[0];
    for item in list {
        if item > largest {
            largest = item;
        }
    }
    largest
}

fn main() {
    let nums = vec![34, 50, 25, 100, 65];
    println!("{}", largest(&nums)); // 100

    let chars = vec!['y', 'm', 'a', 'q'];
    println!("{}", largest(&chars)); // 'y'
}
```

## Struct Genérica

```rust
struct Point<T> {
    x: T,
    y: T,
}

let integer = Point { x: 5, y: 10 };
let float   = Point { x: 1.0, y: 4.0 };
```

## Enum Genérica

```rust
enum Option<T> {
    Some(T),
    None,
}

enum Result<T, E> {
    Ok(T),
    Err(E),
}
```

## Método Genérico

```rust
struct Point<T> { x: T, y: T }

impl<T> Point<T> {
    fn x(&self) -> &T { &self.x }
}

// Only for f64
impl Point<f64> {
    fn distance_from_origin(&self) -> f64 {
        (self.x.powi(2) + self.y.powi(2)).sqrt()
    }
}
```

## Puntos Clave

| Concepto | Significado |
|---------|---------|
| **Monomorfización** | El compilador genera código concreto por tipo usado → costo cero en ejecución. |
| **Parámetro de tipo `T`** | Marcador de posición que se completa en tiempo de compilación. |
| **Múltiples parámetros** | `struct Foo<T, U>` — cada uno puede ser un tipo diferente. |
