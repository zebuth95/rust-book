# Mapas Hash (`HashMap<K, V>`)

Almacena pares clave-valor. No está en el preludio — debe ser importado.

```rust
use std::collections::HashMap;

let mut scores = HashMap::new();
scores.insert(String::from("Blue"), 10);
scores.insert(String::from("Yellow"), 50);
```

## Acceso

```rust
let score = scores.get(&String::from("Blue"));  // Option<&i32>
```

## Iterar

```rust
for (key, value) in &scores {
    println!("{key}: {value}");
}
```

## Actualizar

```rust
scores.insert("Blue".to_string(), 25);                    // sobrescribir
scores.entry("Yellow".to_string()).or_insert(30);         // insertar si no existe
```

> **Nota:** La posesión de las claves/valores se mueve al HashMap al insertar.
