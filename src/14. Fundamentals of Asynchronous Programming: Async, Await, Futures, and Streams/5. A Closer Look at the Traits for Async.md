# A Closer Look at the Traits for Async
The entire async system is build on traits. The **Future trait** is the most fundamental. It contains the associate type
**Output** (the type returned on success) and the core method **poll(self: Pin<&mut Self>, cx: &mut Context<'_>) -> Poll<Self::Output>**

- **Pin and Context**: The use of **Pin** ensures that the **Future** can not be moved in memory after it's polled,
which is critical for self-referential structures (used heavily inside async state machines).
The **Context** provides a mechanism for the **Future** to register a **Waker** so the executor knows when to poll it again.
- **Advanced Insight**: the **async/await** syntax is **syntactic sugar** that automatically generates the state struct and implements
the **Future** trait's **poll** method for you.
