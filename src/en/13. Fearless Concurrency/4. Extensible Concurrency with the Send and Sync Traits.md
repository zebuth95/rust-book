# `Send` & `Sync`

Marker traits for thread safety.

| Trait | Meaning |
|---|---|
| `Send` | Ownership can be transferred across threads |
| `Sync` | Immutable reference (`&T`) can be shared across threads |

- Most types are `Send` and `Sync` automatically.
- `Rc<T>` is NOT `Send`. `RefCell<T>` is NOT `Sync`.
- `Arc<T>` is `Send` + `Sync`. `Mutex<T>` is `Sync`.

> **Key point:** The compiler checks these traits automatically. You rarely need to implement them manually.
