# Vectors (`Vec<T>`)

A growable array of values of the **same type**.

```rust
let mut v: Vec<i32> = Vec::new();
v.push(1);
v.push(2);

// Macro shorthand
let v = vec![1, 2, 3];
```

## Accessing elements

```rust
let v = vec![10, 20, 30];
let third = &v[2];            // panics if out of bounds
let third = v.get(2);         // returns Option<&T>, safe
```

## Iterating

```rust
let mut v = vec![1, 2, 3];
for i in &v { println!("{i}"); }        // read
for i in &mut v { *i += 10; }           // mutate
```

> **Key point:** Use `get()` for safe access. Use indexing when you're sure the index exists.
