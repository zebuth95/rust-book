# Streams: Futures in Sequence
**Stream** is a asynchronous iterator. Just as the **Iterator** trait has a **next()** method that return **Option<Item>**,
the **Stream** trait has a **pull_next()** method that return a **Poll<Option<Item>>**.

Stream represent a series of values arriving over time (network packages, database query results ..).

## Implementation:
Stream are use with an **await**-like syntax often provided by runtime helper traits to process the sequence of values.

## Use Case Example: Processing a Stream of Incremental Values
A common use case for streams is handling events or incremental data fetching over time. This example simulates a process that yields a sequence of numbers every 100 milliseconds.

Example (Simulated Ticker Stream)



    use tokio::time::{sleep, Duration};
    use futures::stream;
    use futures::StreamExt;           
    use tokio_stream::StreamExt as _; // Imports the trait without polluting namespace
    
    async fn process_number_stream() {
    println!("--- Starting Stream Processing ---");
    
        // 1. Create the Stream<Item = impl Future<Output = i32>>
        // UFCS is used here to resolve the `map()` ambiguity
        let number_stream = futures::StreamExt::map(stream::iter(1..=5), |n| async move {
                // Simulate work/delay between items
                sleep(Duration::from_millis(100)).await; 
                n // The item yielded by the future
            })
            .buffer_unordered(1); // Provided by futures::StreamExt
    
        // 2. Process the Stream of Futures
        let mut number_result_stream = number_stream; // Note: mutable binding needed for next()
    
        // tokio::pin! is needed to use the stream in the while loop
        // Using tokio::pin! here on the mutable binding is correct
        tokio::pin!(number_result_stream);
    
        let start_time = tokio::time::Instant::now();
    
        // UFCS is used here to resolve the `next()` ambiguity
        // We must pass a mutable reference to the stream: `&mut number_result_stream`
        while let Some(item) = futures::StreamExt::next(&mut number_result_stream).await {
            let elapsed = start_time.elapsed();
            println!("Received item: {} at T+{:?}", item, elapsed);
        }
        
        println!("--- Stream Processing Complete ---");
    }

    #[tokio::main]
    async fn main() {
        process_number_stream().await;
    }

    Received item: 5 at T+508.479232ms
