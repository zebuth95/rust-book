# `Box<T>` — Heap Allocation

`Box<T>` stores data on the heap.

```rust
let b = Box::new(5);    // 5 on heap, b on stack
println!("{b}");         // 5
```

## Use cases

- Recursive types (unknown size at compile time).
- Large data — transfer ownership without copying.

```rust
enum List {
    Cons(i32, Box<List>),  // Box gives fixed size
    Nil,
}
```

> **Note:** `Box<T>` is dropped automatically when it goes out of scope.
