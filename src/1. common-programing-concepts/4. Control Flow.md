# Control Flow

## `if` / `else if` / `else`

```rust
let n = 7;
if n < 5 {
    println!("small");
} else if n < 10 {
    println!("medium");
} else {
    println!("large");
}
```

`if` is an expression — can assign result:

```rust
let condition = true;
let n = if condition { 5 } else { 6 };
```

## Loops

### `loop` — infinite, can `break` with value

```rust
let mut counter = 0;
let result = loop {
    counter += 1;
    if counter == 10 {
        break counter * 2;
    }
};
println!("{result}");  // 20
```

### `while` — conditional

```rust
let mut n = 3;
while n != 0 {
    println!("{n}");
    n -= 1;
}
```

### `for` — iterate over collection (preferred)

```rust
let a = [10, 20, 30];
for elem in a {
    println!("{elem}");
}

// Range with rev
for n in (1..4).rev() {
    println!("{n}");
}
```

> **Key point:** Use `for` over arrays/iterators — safer and faster than index-based loops.
