# Refutabilidad

| Patrón | ¿Refutable? | Usado en |
|---|---|---|
| `let x = ...` | Irrefutable (siempre coincide) | Sentencias `let` |
| `if let Some(x) = ...` | Refutable (podría fallar) | `if let`, `match` |
| `match` arms | Ambos | Expresiones `match` |

```rust
let x = 5;                     // irrefutable ✅
let Some(x) = some_value;      // ❌ error: refutable in let
if let Some(x) = some_value {} // ✅ refutable here
```

> **Punto clave:** `let` solo acepta patrones irrefutables. Usa `if let` o `match` para patrones refutables.
