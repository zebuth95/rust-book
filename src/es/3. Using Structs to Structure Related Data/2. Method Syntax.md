# Sintaxis de Métodos

Los métodos son funciones definidas **dentro de un bloque `impl`**, asociadas a un tipo.

```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }

    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width > other.width && self.height > other.height
    }
}

fn main() {
    let r = Rectangle { width: 30, height: 50 };
    println!("Area: {}", r.area());
}
```

## Funciones Asociadas (sin `self`)

Se usan como constructores. Se invocan con `::`.

```rust
impl Rectangle {
    fn square(size: u32) -> Self {
        Self { width: size, height: size }
    }
}

let sq = Rectangle::square(10);
```

> **Nota:** `Self` = el tipo después de `impl`. Un struct puede tener múltiples bloques `impl`.
