# Closures

Los closures son funciones anónimas que pueden **capturar variables** de su ámbito.

```rust
let x = 5;
let add = |y| y + x;    // captures x immutably
println!("{}", add(3));  // 8
```

## Bounds de Trait

| Trait | Captura | Se llama |
|---|---|---|
| `Fn` | Préstamo inmutable (`&T`) | Muchas veces |
| `FnMut` | Préstamo mutable (`&mut T`) | Muchas veces |
| `FnOnce` | Movimiento/consumo (`T`) | Una vez |

## La palabra clave `move`

Fuerza al closure a tomar posesión de las variables capturadas (requerido al lanzar hilos).

```rust
let data = vec![1, 2, 3];
let closure = move || {
    println!("{:?}", data);  // data moved here
};
closure();
// println!("{:?}", data);  // ❌ data moved
```

> **Punto clave:** Los closures infieren su trait según lo que hacen con las variables capturadas.
