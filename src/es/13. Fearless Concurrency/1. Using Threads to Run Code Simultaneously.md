# Hilos (Threads)

Crea un hilo con `std::thread::spawn`.

```rust
use std::thread;
use std::time::Duration;

let handle = thread::spawn(|| {
    for i in 1..5 {
        println!("thread: {i}");
        thread::sleep(Duration::from_millis(1));
    }
});

handle.join().unwrap();  // esperar a que el hilo termine
```

## Cláusulas `move` con hilos

```rust
let v = vec![1, 2, 3];
let handle = thread::spawn(move || {
    println!("{:?}", v);  // v se mueve al hilo
});
handle.join().unwrap();
```

> **Nota:** Usa `move` para transferir la propiedad de los datos capturados al hilo.
