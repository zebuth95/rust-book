# Release Profiles

Two default profiles:

| Profile | Command | Optimization | Use |
|---|---|---|---|
| `dev` | `cargo build` | None (`opt-level = 0`) | Fast compilation |
| `release` | `cargo build --release` | Max (`opt-level = 3`) | Production |

Customize in `Cargo.toml`:

```toml
[profile.dev]
opt-level = 0

[profile.release]
opt-level = 3
```

> **Note:** `cargo build --release` is slower to compile but produces faster code.
