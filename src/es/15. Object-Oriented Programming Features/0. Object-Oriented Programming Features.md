# Características de POO en Rust

Rust no es un lenguaje POO tradicional pero soporta principios de POO a través de:

| Concepto POO | Mecanismo de Rust |
|---|---|
| Encapsulación | Campos `pub` / privados |
| Polimorfismo | Traits + Genéricos (estático) / Trait Objects (dinámico) |
| Herencia | **No soportada** — usa composición y traits en su lugar |
