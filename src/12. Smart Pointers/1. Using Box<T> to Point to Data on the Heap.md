# Using Box<T> to Point to Data on the Heap
**Box<T>** is the simplest smart pointer. Its primary use is to allocate data on the head instead of the stack.

### Use cases:

- When you have a type whose size is unknown at compile time (like recursive types) and need a fixed size wrapper.
- When you have a large amount of data and want to transfer ownership without copying it.

### Mechanism 
When a **Box<T>** goes out of scope, the head data it points to is automatically deallocated.

    // 'b' is on the stack, but the integer '5' is allocated on the heap.
    let b = Box::new(5);
    println!("b = {}", b); // Accesses the value on the heap
