# Hash Maps (`HashMap<K, V>`)

Stores key-value pairs. Not in the prelude — must import.

```rust
use std::collections::HashMap;

let mut scores = HashMap::new();
scores.insert(String::from("Blue"), 10);
scores.insert(String::from("Yellow"), 50);
```

## Access

```rust
let score = scores.get(&String::from("Blue"));  // Option<&i32>
```

## Iterate

```rust
for (key, value) in &scores {
    println!("{key}: {value}");
}
```

## Update

```rust
scores.insert("Blue".to_string(), 25);                    // overwrite
scores.entry("Yellow".to_string()).or_insert(30);         // insert if missing
```

> **Note:** Ownership of keys/values moves into the HashMap on insert.
