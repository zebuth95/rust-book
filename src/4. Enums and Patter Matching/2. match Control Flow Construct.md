# `match` Control Flow

`match` compares a value against patterns. **Must be exhaustive**.

```rust
enum Coin { Penny, Nickel, Dime, Quarter }

fn value_in_cents(coin: Coin) -> u8 {
    match coin {
        Coin::Penny => 1,
        Coin::Nickel => 5,
        Coin::Dime => 10,
        Coin::Quarter => 25,
    }
}
```

## Matching with `Option<T>`

```rust
fn plus_one(x: Option<i32>) -> Option<i32> {
    match x {
        None => None,
        Some(i) => Some(i + 1),
    }
}

let six = plus_one(Some(5));
let none = plus_one(None);
```

## Catch-all with `_`

```rust
let dice = 9;
match dice {
    3 => add_fancy_hat(),
    7 => remove_fancy_hat(),
    _ => (),   // do nothing for everything else
}
```

> **Key point:** Every possible value must be handled. Use `_` as a wildcard.
