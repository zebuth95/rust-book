# Generic Data Types

Write functions, structs, enums, and methods that work with **any type** — no code duplication.

## Generic Function

```rust
fn largest<T: std::cmp::PartialOrd>(list: &[T]) -> &T {
    let mut largest = &list[0];
    for item in list {
        if item > largest {
            largest = item;
        }
    }
    largest
}

fn main() {
    let nums = vec![34, 50, 25, 100, 65];
    println!("{}", largest(&nums)); // 100

    let chars = vec!['y', 'm', 'a', 'q'];
    println!("{}", largest(&chars)); // 'y'
}
```

## Generic Struct

```rust
struct Point<T> {
    x: T,
    y: T,
}

let integer = Point { x: 5, y: 10 };
let float   = Point { x: 1.0, y: 4.0 };
```

## Generic Enum

```rust
enum Option<T> {
    Some(T),
    None,
}

enum Result<T, E> {
    Ok(T),
    Err(E),
}
```

## Generic Method

```rust
struct Point<T> { x: T, y: T }

impl<T> Point<T> {
    fn x(&self) -> &T { &self.x }
}

// Only for f64
impl Point<f64> {
    fn distance_from_origin(&self) -> f64 {
        (self.x.powi(2) + self.y.powi(2)).sqrt()
    }
}
```

---

## Key Points

| Concept | Meaning |
|---------|---------|
| **Monomorphization** | Compiler generates concrete code per type used → zero runtime cost. |
| **Type param `T`** | Placeholder filled in at compile time. |
| **Multiple params** | `struct Foo<T, U>` — each can be a different type. |
