# Running Tests

```bash
cargo test                         # run all tests
cargo test test_name               # filter by name
cargo test -- --test-threads=1     # single-threaded (sequential)
cargo test -- --nocapture          # show println! output
cargo test -- --ignored            # run only ignored tests
```

## Ignoring tests

```rust
#[test]
#[ignore]
fn expensive_test() { /* ... */ }
```

> **Note:** Tests run in parallel by default. Use `--test-threads=1` to serialize.
