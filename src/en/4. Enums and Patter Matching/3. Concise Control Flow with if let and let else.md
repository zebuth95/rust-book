# `if let` & `let else`

Shorter syntax when you only care about **one pattern**.

## `if let`

```rust
let config_max = Some(3u8);

// Instead of:
// match config_max {
//     Some(max) => println!("{max}"),
//     _ => (),
// }

if let Some(max) = config_max {
    println!("{max}");
}
```

## `if let` with `else`

```rust
let mut count = 0;
if let Coin::Quarter(state) = coin {
    println!("{state:?}");
} else {
    count += 1;
}
```

## `let...else` (happy path)

```rust
fn describe(coin: Coin) -> Option<String> {
    let Coin::Quarter(state) = coin else {
        return None;   // early return on mismatch
    };
    Some(format!("{state:?}"))
}
```

> **Note:** Use `if let` when you only need one arm. Use `match` when you need exhaustiveness.
