# Rutas para Referirse a un Elemento en el Árbol de Módulos
Una **ruta (path)** es la forma en que te refieres a un elemento en el árbol de módulos.
Existen dos tipos de rutas.

- Una **ruta absoluta** comienza desde la raíz del crate (`crate::`).
- Una **ruta relativa** comienza desde el módulo actual (`super::` o `self::`).
- Las rutas están compuestas por uno o más identificadores separados por dobles dos puntos (`::`).


    // En src/main.rs
    fn main() {
        // Ruta absoluta, comenzando desde la raíz del crate
        let plant = my_project::garden::vegetables::Asparagus {
            name: String::from("Spargel"),
        };
    
        // Nota: Una ruta relativa se usaría dentro del mismo crate,
        // por ejemplo, si estuvieras en un módulo diferente dentro de `my_project`.
    }
