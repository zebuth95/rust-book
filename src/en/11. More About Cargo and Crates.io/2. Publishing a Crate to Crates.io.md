# Publishing to Crates.io

```bash
cargo login           # save API token
cargo publish         # publish current crate
cargo publish --dry-run   # check without publishing
```

## Requirements

Add metadata to `Cargo.toml`:

```toml
[package]
license = "MIT"
description = "A useful crate"
repository = "https://github.com/user/crate"
```

> **Important:** Published versions are **permanent** — can't be deleted or modified.
