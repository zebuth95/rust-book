# Advanced Types

## Newtype Pattern

```rust
struct Meters(f64);
struct Kilometers(f64);

impl Meters {
    fn to_kilometers(&self) -> Kilometers { Kilometers(self.0 / 1000.0) }
}
```

## Type Aliases

```rust
type Kilometers = i32;
type Thunk = Box<dyn Fn() + Send + 'static>;
```

## Never Type (`!`)

A function that never returns:

```rust
fn forever() -> ! {
    loop { /* ... */ }
}
```

## Dynamically Sized Types (`?Sized`)

```rust
fn generic<T: ?Sized>(t: &T) {
    // T can be unsized (e.g., str, [u8])
}
```

> **Key point:** `!` is the never type — used in `continue`, `panic!`, infinite loops.
