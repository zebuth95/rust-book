# `RefCell<T>` — Mutabilidad Interior

Permite la mutación a través de una referencia inmutable. Las reglas de préstamo se verifican en **tiempo de ejecución** (no en tiempo de compilación).

```rust
use std::cell::RefCell;

let x = RefCell::new(String::from("Hello"));
let mut s = x.borrow_mut();   // préstamo mutable
s.push_str(" World");
// s se libera aquí
println!("{}", x.borrow());    // "Hello World"
```

## Reglas

- `borrow()` → múltiples préstamos inmutables permitidos.
- `borrow_mut()` → solo un préstamo mutable a la vez (entra en pánico si se viola en tiempo de ejecución).

## Patrón común: `Rc<RefCell<T>>`

Comparte y muta datos en código monohilo.

```rust
let data = Rc::new(RefCell::new(5));
*data.borrow_mut() += 1;
```

> **Punto clave:** Usa `RefCell<T>` cuando el compilador no pueda verificar la seguridad pero tú sepas que es seguro.
