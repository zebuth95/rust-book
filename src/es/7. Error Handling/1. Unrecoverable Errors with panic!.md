# `panic!` — Errores Irrecuperables

`panic!` imprime un mensaje, desenrolla la pila y termina la ejecución.

```rust
fn main() {
    let v = vec![1, 2, 3];
    v[99];  // panic: index out of bounds
}
```

```
thread 'main' panicked at src/main.rs:4:6:
index out of bounds: len 3, index 99
```

Usa `RUST_BACKTRACE=1` para ver la pila de llamadas completa.

> **Nota:** Usa `panic!` para bugs, aserciones fallidas o estados imposibles.
