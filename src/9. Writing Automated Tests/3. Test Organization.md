# Test Organization

## Unit tests

Test one function in isolation. Placed in the same file, inside `mod tests`.

```rust
pub fn add_two(a: i32) -> i32 { a + 2 }

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn it_adds_two() {
        assert_eq!(4, add_two(2));
    }
}
```

## Integration tests

Test the public API. Placed in `tests/` directory (next to `src/`).

```rust
// tests/integration_test.rs
use my_crate;

#[test]
fn it_adds_two() {
    assert_eq!(4, my_crate::add_two(2));
}
```

> **Note:** Unit tests can test `private` functions. Integration tests cannot.
