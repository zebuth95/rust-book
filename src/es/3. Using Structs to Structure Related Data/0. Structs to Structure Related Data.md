# Structs

Un **struct** agrupa datos relacionados.

## Definición e Instanciación

```rust
struct User {
    username: String,
    email: String,
    sign_in_count: u64,
    active: bool,
}

fn main() {
    let user1 = User {
        email: String::from("a@b.com"),
        username: String::from("alice"),
        active: true,
        sign_in_count: 1,
    };
}
```

## Atajo de Inicialización de Campos

```rust
fn build_user(email: String, username: String) -> User {
    User {
        email,      // shorthand for email: email
        username,   // shorthand for username: username
        active: true,
        sign_in_count: 1,
    }
}
```

## Actualización desde otra instancia

```rust
let user2 = User {
    email: String::from("new@b.com"),
    ..user1       // inherit rest from user1
};
```

## Structs de Tupla

```rust
struct Color(i32, i32, i32);
let black = Color(0, 0, 0);
```

## Struct Tipo Unidad (Unit-Like)

```rust
struct AlwaysEqual;  // no fields
```
