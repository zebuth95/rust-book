# Enums

An **enum** defines a type by enumerating its possible **variants**.

```rust
enum IpAddr {
    V4(u8, u8, u8, u8),
    V6(String),
}

let home = IpAddr::V4(127, 0, 0, 1);
let loopback = IpAddr::V6(String::from("::1"));
```

## Enum with different payloads

```rust
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}
```

## `Option<T>` — no nulls in Rust

```rust
enum Option<T> {
    None,
    Some(T),
}

let some_num = Some(5);
let absent: Option<i32> = None;
```

You must handle `Option` explicitly — the compiler won't let you use a value that might be `None`.

> **Key point:** `Option<T>` replaces `null`. No more null pointer exceptions.
