# Refutability

| Pattern | Refutable? | Used in |
|---|---|---|
| `let x = ...` | Irrefutable (always matches) | `let` statements |
| `if let Some(x) = ...` | Refutable (might fail) | `if let`, `match` |
| `match` arms | Both | `match` expressions |

```rust
let x = 5;                     // irrefutable ✅
let Some(x) = some_value;      // ❌ error: refutable in let
if let Some(x) = some_value {} // ✅ refutable here
```

> **Key point:** `let` only accepts irrefutable patterns. Use `if let` or `match` for refutable patterns.
