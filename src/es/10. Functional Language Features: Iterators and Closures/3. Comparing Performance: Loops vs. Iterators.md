# Rendimiento: Bucles vs Iteradores

**Los iteradores son tan rápidos como los bucles escritos a mano.** El compilador inlinea y elimina el overhead.

```rust
// Manual loop
let mut sum = 0;
for i in 1..=1000 {
    sum += i;
}

// Iterator (same performance)
let sum: i32 = (1..=1000).sum();
```

> **Nota:** Las abstracciones de costo cero de Rust significan que no sacrificas rendimiento por legibilidad. Prefiere iteradores para mayor claridad.
