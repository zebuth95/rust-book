# To `panic!` or Not to `panic!`

| Use `panic!` when... | Use `Result` when... |
|---|---|
| A bug or impossible state | Failure is expected |
| Examples, prototypes, tests | Library code |
| You know the result is `Ok` | Caller should decide how to handle |

**Rule of thumb:** Return `Result` from library functions. Let the caller decide.

> **Note:** Use `.unwrap()` and `.expect()` sparingly — only when you're certain it won't fail.
