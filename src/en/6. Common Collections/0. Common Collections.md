# Common Collections

Collections store multiple values on the **heap** (dynamic size).

| Type | Description |
|---|---|
| `Vec<T>` | Growable array of same-type values |
| `String` | UTF-8 text (wraps `Vec<u8>`) |
| `HashMap<K,V>` | Key-value map |

> **Note:** Arrays & tuples are stack-allocated. Collections are heap-allocated.
