# Publicar en Crates.io

```bash
cargo login           # guardar token API
cargo publish         # publicar el crate actual
cargo publish --dry-run   # verificar sin publicar
```

## Requisitos

Agrega metadatos a `Cargo.toml`:

```toml
[package]
license = "MIT"
description = "Un crate útil"
repository = "https://github.com/usuario/crate"
```

> **Importante:** Las versiones publicadas son **permanentes** — no se pueden eliminar ni modificar.
