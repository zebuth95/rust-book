# Functions

Defined with `fn`. Convention: `snake_case`.

```rust
fn main() {
    greet("World");
}

fn greet(name: &str) {
    println!("Hello, {name}!");
}
```

## Parameters & Return Values

```rust
fn add(x: i32, y: i32) -> i32 {
    x + y   // no semicolon = return expression
}
```

## Statements vs Expressions

- **Statements** — perform an action, return nothing (`let x = 5;`)
- **Expressions** — evaluate to a value (`x + 1`, function calls)

```rust
fn main() {
    let y = {
        let x = 3;
        x + 1    // expression, returns 4
    };
    println!("{y}");  // 4
}
```

> **Key point:** Last expression in a function body is the return value (no `return` needed).
