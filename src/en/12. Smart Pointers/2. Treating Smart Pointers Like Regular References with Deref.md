# `Deref` Trait

Makes smart pointers work like regular references with the `*` operator.

```rust
let x = 5;
let y = Box::new(x);
assert_eq!(5, *y);   // *y works because Box implements Deref
```

## Deref coercion

Rust automatically converts `&Box<T>` to `&T` (and beyond).

```rust
fn hello(name: &str) { println!("{name}"); }

let m = Box::new(String::from("Rust"));
hello(&m);  // &Box<String> -> &String -> &str (automatic)
```

> **Key point:** Deref coercion makes smart pointers ergonomic to use.
