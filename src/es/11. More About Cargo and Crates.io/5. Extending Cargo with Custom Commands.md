# Comandos Personalizados de Cargo

Cualquier binario llamado `cargo-algo` en `PATH` puede ejecutarse como `cargo algo`.

```bash
# Instalar un comando personalizado
cargo install cargo-watch

# Ejecutarlo
cargo watch
```

Ejemplos populares: `cargo clippy` (lint), `cargo audit` (verificador de seguridad).

> **Nota:** Así funciona el sistema de extensibilidad de Cargo — es trivial agregar subcomandos personalizados.
