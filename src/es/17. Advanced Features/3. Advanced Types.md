# Tipos Avanzados

## Patrón Newtype

```rust
struct Meters(f64);
struct Kilometers(f64);

impl Meters {
    fn to_kilometers(&self) -> Kilometers { Kilometers(self.0 / 1000.0) }
}
```

## Alias de Tipo

```rust
type Kilometers = i32;
type Thunk = Box<dyn Fn() + Send + 'static>;
```

## Never Type (`!`)

Una función que nunca retorna:

```rust
fn forever() -> ! {
    loop { /* ... */ }
}
```

## Tipos de Tamaño Dinámico (`?Sized`)

```rust
fn generic<T: ?Sized>(t: &T) {
    // T can be unsized (e.g., str, [u8])
}
```

> **Punto clave:** `!` es el never type — se usa en `continue`, `panic!`, bucles infinitos.

#### Const Generics

Parámetros genéricos que son **constantes en tiempo de compilación** (no tipos). Estabilizados incrementalmente desde Rust 1.51.

```rust
struct Array<T, const N: usize> {
    data: [T; N],
}

impl<T: std::fmt::Debug, const N: usize> Array<T, N> {
    fn print(&self) {
        println!("{:?}", self.data);
    }
}

let small: Array<i32, 3> = Array { data: [1, 2, 3] };
let large: Array<i32, 100> = Array { data: [0; 100] };
```

Los const generics se pueden usar con:
- Tamaños de array (`[T; N]`)
- Cualquier `usize`, `u8`, `i32`, `bool`, `char` u otros tipos enteros
- Evaluación de expresiones en tiempo de compilación

```rust
fn dot_product<const N: usize>(a: [f64; N], b: [f64; N]) -> f64 {
    let mut sum = 0.0;
    for i in 0..N { sum += a[i] * b[i]; }
    sum
}
```

> **Punto clave:** Los const generics eliminan las comprobaciones de tamaño en tiempo de ejecución y permiten estructuras de datos de tamaño fijo type-safe.
