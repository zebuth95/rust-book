# Vectores (`Vec<T>`)

Un arreglo redimensionable de valores del **mismo tipo**.

```rust
let mut v: Vec<i32> = Vec::new();
v.push(1);
v.push(2);

// Macro abreviada
let v = vec![1, 2, 3];
```

## Accediendo a elementos

```rust
let v = vec![10, 20, 30];
let third = &v[2];            // entra en pánico si está fuera de límites
let third = v.get(2);         // devuelve Option<&T>, seguro
```

## Iterando

```rust
let mut v = vec![1, 2, 3];
for i in &v { println!("{i}"); }        // lectura
for i in &mut v { *i += 10; }           // mutación
```

> **Punto clave:** Usa `get()` para acceso seguro. Usa indexación cuando estés seguro de que el índice existe.
