# Control de Flujo

## `if` / `else if` / `else`

```rust
let n = 7;
if n < 5 {
    println!("small");
} else if n < 10 {
    println!("medium");
} else {
    println!("large");
}
```

`if` es una expresión — puede asignar un resultado:

```rust
let condition = true;
let n = if condition { 5 } else { 6 };
```

## Bucles

### `loop` — infinito, puede hacer `break` con valor

```rust
let mut counter = 0;
let result = loop {
    counter += 1;
    if counter == 10 {
        break counter * 2;
    }
};
println!("{result}");  // 20
```

### `while` — condicional

```rust
let mut n = 3;
while n != 0 {
    println!("{n}");
    n -= 1;
}
```

### `for` — iterar sobre colecciones (preferido)

```rust
let a = [10, 20, 30];
for elem in a {
    println!("{elem}");
}

// Range with rev
for n in (1..4).rev() {
    println!("{n}");
}
```

> **Punto clave:** Usa `for` sobre arreglos/iteradores — más seguro y rápido que bucles basados en índices.

## `while let` — Bucle Condicional con Patrón

`while let` se ejecuta repetidamente mientras un patrón coincida.

```rust
let mut stack = vec![1, 2, 3];
while let Some(top) = stack.pop() {
    println!("{top}");
}
```

## Etiquetas de Bucles

Las etiquetas desambiguan a qué bucle se aplica un `break` o `continue` cuando los bucles están anidados.

```rust
'outer: for i in 0..3 {
    for j in 0..3 {
        if i == 1 && j == 1 {
            break 'outer;
        }
        println!("({i}, {j})");
    }
}
```

- Las etiquetas también se pueden usar con `continue 'outer;` para saltar a la siguiente iteración del bucle externo.
