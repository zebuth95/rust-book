# How to Write Tests

## Assertion macros

```rust
#[test]
fn check_addition() {
    assert!(2 + 2 == 4);
    assert_eq!(2 + 2, 4);
    assert_ne!(2 + 2, 5);
}
```

## Custom failure messages

```rust
assert_eq!(2 + 2, 4, "2 + 2 should be 4");
```

## Expecting panics

```rust
#[test]
#[should_panic(expected = "out of bounds")]
fn index_out_of_bounds() {
    let v = vec![1, 2];
    v[10];
}
```

## Using `Result<T, E>` in tests

```rust
#[test]
fn it_works() -> Result<(), String> {
    if 2 + 2 == 4 { Ok(()) } else { Err("nope".into()) }
}
```

> **Note:** `#[should_panic]` checks that the code panics with a matching message.
