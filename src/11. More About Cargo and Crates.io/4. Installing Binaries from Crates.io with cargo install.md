# Installing Binaries from Crates.io with cargo install
The cargo install command is used to compile and install executable binary crates from Crates.io directly to your local system. It's the primary way to distribute Rust command-line tools to end-users.

Result: The resulting binary is typically placed in a directory like ~/.cargo/bin, which should be on your system's PATH.
 
    # Downloads, compiles, and installs the 'rg' (ripgrep) binary to your PATH
    cargo install ripgrep
