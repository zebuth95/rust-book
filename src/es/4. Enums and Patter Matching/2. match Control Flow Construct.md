# Flujo de Control `match`

`match` compara un valor contra patrones. **Debe ser exhaustivo**.

```rust
enum Coin { Penny, Nickel, Dime, Quarter }

fn value_in_cents(coin: Coin) -> u8 {
    match coin {
        Coin::Penny => 1,
        Coin::Nickel => 5,
        Coin::Dime => 10,
        Coin::Quarter => 25,
    }
}
```

## Coincidencia con `Option<T>`

```rust
fn plus_one(x: Option<i32>) -> Option<i32> {
    match x {
        None => None,
        Some(i) => Some(i + 1),
    }
}

let six = plus_one(Some(5));
let none = plus_one(None);
```

## Caso comodín con `_`

```rust
let dice = 9;
match dice {
    3 => add_fancy_hat(),
    7 => remove_fancy_hat(),
    _ => (),   // do nothing for everything else
}
```

> **Punto clave:** Cada valor posible debe ser manejado. Usa `_` como comodín.
