# `if let` y `let else`

Sintaxis más corta cuando solo te interesa **un patrón**.

## `if let`

```rust
let config_max = Some(3u8);

// Instead of:
// match config_max {
//     Some(max) => println!("{max}"),
//     _ => (),
// }

if let Some(max) = config_max {
    println!("{max}");
}
```

## `if let` con `else`

```rust
let mut count = 0;
if let Coin::Quarter(state) = coin {
    println!("{state:?}");
} else {
    count += 1;
}
```

## `let...else` (camino feliz)

```rust
fn describe(coin: Coin) -> Option<String> {
    let Coin::Quarter(state) = coin else {
        return None;   // early return on mismatch
    };
    Some(format!("{state:?}"))
}
```

> **Nota:** Usa `if let` cuando solo necesites un brazo. Usa `match` cuando necesites exhaustividad.
