# Performance: Loops vs Iterators

**Iterators are as fast as hand-written loops.** The compiler inlines and eliminates overhead.

```rust
// Manual loop
let mut sum = 0;
for i in 1..=1000 {
    sum += i;
}

// Iterator (same performance)
let sum: i32 = (1..=1000).sum();
```

> **Note:** Rust's zero-cost abstractions mean you don't sacrifice performance for readability. Prefer iterators for clarity.
