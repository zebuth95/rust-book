# Threads

Spawn a thread with `std::thread::spawn`.

```rust
use std::thread;
use std::time::Duration;

let handle = thread::spawn(|| {
    for i in 1..5 {
        println!("thread: {i}");
        thread::sleep(Duration::from_millis(1));
    }
});

handle.join().unwrap();  // wait for thread to finish
```

## `move` closures with threads

```rust
let v = vec![1, 2, 3];
let handle = thread::spawn(move || {
    println!("{:?}", v);  // v moved into thread
});
handle.join().unwrap();
```

> **Note:** Use `move` to transfer ownership of captured data into the thread.
