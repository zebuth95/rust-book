# Enums

Un **enum** define un tipo enumerando sus posibles **variantes**.

```rust
enum IpAddr {
    V4(u8, u8, u8, u8),
    V6(String),
}

let home = IpAddr::V4(127, 0, 0, 1);
let loopback = IpAddr::V6(String::from("::1"));
```

## Enum con diferentes cargas (payloads)

```rust
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}
```

## `Option<T>` — sin null en Rust

```rust
enum Option<T> {
    None,
    Some(T),
}

let some_num = Some(5);
let absent: Option<i32> = None;
```

Debes manejar `Option` explícitamente — el compilador no te permitirá usar un valor que podría ser `None`.

> **Punto clave:** `Option<T>` reemplaza a `null`. No más excepciones de puntero nulo.
