# Storing Lists of Values with Vectors
A **vector** is a dynamic, contiguous, growable array-like structure.
It stores elements of the same type on the heap. Accessing elements by index is an O(1) operation.
When a vector's capacity is exceeded, it relocates its data to a new, larger area in memory, copies the old elements, and drops the original.
This reallocation is a key performance consideration.

Creating a vector and pushing elements:

    let mut v = Vec::new(); // Or let v = vec![1, 2, 3];
    v.push(5);
    v.push(6);
    v.push(7);

Accessing elements:

    // Using indexing (can panic if out of bounds)
    let third: &i32 = &v[2];
    println!("The third element is {}", third);
    
    // Using `get` (returns `Option<&T>`, which is safe)
    match v.get(2) {
        Some(third) => println!("The third element is {}", third),
        None => println!("There is no third element."),
    }
