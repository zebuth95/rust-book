# Strings

Rust has two string types:

- `&str` — immutable string slice (view into data)
- `String` — growable, heap-allocated UTF-8 text

```rust
let mut s = String::new();
let s = "hello".to_string();
let s = String::from("hello");
```

## Updating

```rust
let mut s = String::from("foo");
s.push_str("bar");         // "foobar"
s.push('!');               // "foobar!"

let s1 = String::from("Hello, ");
let s2 = String::from("world!");
let s3 = s1 + &s2;         // note: s1 is MOVED here

let s = format!("{s2}-{s3}");  // doesn't take ownership
```

## No indexing!

Rust strings don't support `s[0]` — UTF-8 characters can be multiple bytes.

```rust
for c in "Зд".chars() { println!("{c}"); }   // З, д
for b in "Зд".bytes() { println!("{b}"); }   // 208, 151, 208, 180
```

> **Note:** Use `chars()` or `bytes()` for iteration, never direct indexing.
