# `panic!` — Unrecoverable Errors

`panic!` prints a message, unwinds the stack, and exits.

```rust
fn main() {
    let v = vec![1, 2, 3];
    v[99];  // panic: index out of bounds
}
```

```
thread 'main' panicked at src/main.rs:4:6:
index out of bounds: len 3, index 99
```

Use `RUST_BACKTRACE=1` to see the full call stack.

> **Note:** Use `panic!` for bugs, failed assertions, or impossible states.
