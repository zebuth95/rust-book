# Separando Módulos en Diferentes Archivos

- Esta sección aclara aún más el uso de la palabra clave `use`. Se discute cómo manejar casos donde dos elementos tienen el mismo nombre. Puedes usar un alias con la palabra clave `as` para darle a uno de los elementos un nombre diferente, por ejemplo `use std::io::Result as IoResult;`.

- La combinación `pub use` se utiliza para re-exportar. Esto hace que un elemento sea público para que el código fuera del módulo pueda usar el elemento a través de la ruta de tu módulo.

- El operador glob `*` se puede usar para traer todos los elementos públicos definidos en una ruta al ámbito actual.


    // En un módulo
    mod back_of_house {
        pub struct Breakfast {
            pub toast: String,         // Campo público
            seasonal_fruit: String,    // Campo privado por defecto
        }
    }

En este ejemplo, `toast` puede ser accedido por otros módulos, pero `seasonal_fruit` no.
