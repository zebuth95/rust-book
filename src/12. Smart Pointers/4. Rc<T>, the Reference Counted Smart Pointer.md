# `Rc<T>` — Reference Counting

Multiple ownership in **single-threaded** code.

```rust
use std::rc::Rc;

let a = Rc::new(String::from("shared"));
println!("{}", Rc::strong_count(&a));    // 1

let b = Rc::clone(&a);                   // cheap: increments count
println!("{}", Rc::strong_count(&a));    // 2
```

- Data freed when count reaches 0.
- `Rc::clone` is cheap — just increments a counter.

> **Warning:** Use `Arc<T>` (atomic) for multi-threaded code. `Rc<T>` is not `Send`.
