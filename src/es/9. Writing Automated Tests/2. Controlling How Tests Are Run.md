# Ejecutando Pruebas

```bash
cargo test                         # ejecutar todas las pruebas
cargo test test_name               # filtrar por nombre
cargo test -- --test-threads=1     # un solo hilo (secuencial)
cargo test -- --nocapture          # mostrar salida de println!
cargo test -- --ignored            # ejecutar solo pruebas ignoradas
```

## Ignorando pruebas

```rust
#[test]
#[ignore]
fn expensive_test() { /* ... */ }
```

> **Nota:** Las pruebas se ejecutan en paralelo por defecto. Usa `--test-threads=1` para serializar.
