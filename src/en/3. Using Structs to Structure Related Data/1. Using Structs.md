# Using Structs — Example

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

fn main() {
    let rect = Rectangle { width: 30, height: 50 };
    println!("Area: {}", area(&rect));
}

fn area(r: &Rectangle) -> u32 {
    r.width * r.height
}
```

> **Note:** Borrow the struct (`&Rectangle`) to avoid moving ownership.
