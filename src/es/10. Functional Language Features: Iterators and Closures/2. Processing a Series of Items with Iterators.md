# Iteradores

Los iteradores son **perezosos** — no hacen nada hasta que se consumen.

```rust
let v1 = vec![1, 2, 3];

let iter = v1.iter();            // lazy, nothing happens
let sum: i32 = iter.sum();       // consuming adapter, drives iteration
```

## El Trait `Iterator`

```rust
pub trait Iterator {
    type Item;
    fn next(&mut self) -> Option<Self::Item>;
    // many default methods provided...
}
```

## Método `next()`

```rust
let mut iter = vec![1, 2].iter();
assert_eq!(iter.next(), Some(&1));
assert_eq!(iter.next(), Some(&2));
assert_eq!(iter.next(), None);
```

## Tres Formas de Iteración

| Método | Retorna | Propiedad |
|--------|---------|-----------|
| `.iter()` | `&T` | Préstamo (solo lectura) |
| `.iter_mut()` | `&mut T` | Préstamo (mutable) |
| `.into_iter()` | `T` | Consume (toma posesión) |

```rust
let v = vec![1, 2, 3];

for x in &v { }          // implicit .iter()
for x in &mut v { }      // implicit .iter_mut()
for x in v { }           // implicit .into_iter(), v consumed
```

## Adaptadores Comunes (No Consumidores)

Estos retornan un nuevo iterador (perezoso):

```rust
let v = vec![1, 2, 3, 4, 5];

let mapped: Vec<_> = v.iter().map(|x| x * 2).collect();
let filtered: Vec<_> = v.iter().filter(|&&x| x % 2 == 0).collect();
let skipped: Vec<_> = v.iter().skip(2).collect();
let taken: Vec<_> = v.iter().take(3).collect();
let folded: i32 = v.iter().fold(0, |acc, x| acc + x);
```

## Adaptadores Consumidores

Estos llevan el iterador hasta completarse:

```rust
let v = vec![1, 2, 3];
let sum: i32 = v.iter().sum();
let count = v.iter().count();
let max = v.iter().max();
let min = v.iter().min();
let collected: Vec<_> = v.iter().collect();
let any = v.iter().any(|&x| x > 2);
let all = v.iter().all(|&x| x > 0);
```

## Encadenamiento

```rust
let v = vec![3, 1, 4, 1, 5];
let result: Vec<_> = v.iter()
    .filter(|&&x| x > 2)
    .map(|x| x * 10)
    .collect();  // [30, 40, 50]
```

## Trait `IntoIterator`

Hace que un tipo sea iterable con bucles `for`:

```rust
pub trait IntoIterator {
    type Item;
    type IntoIter: Iterator<Item = Self::Item>;
    fn into_iter(self) -> Self::IntoIter;
}
```

Cualquier tipo que implemente `IntoIterator` funciona en `for x in value { }`.

## Trait `FromIterator`

Crea una colección a partir de un iterador (usado por `.collect()`):

```rust
pub trait FromIterator<A> {
    fn from_iter<T: IntoIterator<Item = A>>(iter: T) -> Self;
}
```

## Implementación Personalizada de Iterator

```rust
struct Counter {
    count: u32,
}

impl Counter {
    fn new() -> Self { Counter { count: 0 } }
}

impl Iterator for Counter {
    type Item = u32;

    fn next(&mut self) -> Option<Self::Item> {
        self.count += 1;
        if self.count < 6 { Some(self.count) } else { None }
    }
}

let sum: u32 = Counter::new()
    .zip(Counter::new().skip(1))
    .map(|(a, b)| a * b)
    .filter(|x| x % 3 == 0)
    .sum();
assert_eq!(sum, 18);
```

> **Punto clave:** Los iteradores son de costo cero — se optimizan al mismo código que los bucles escritos a mano.
