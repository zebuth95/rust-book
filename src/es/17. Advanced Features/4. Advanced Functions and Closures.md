# Funciones y Closures Avanzados

## Punteros a Función

```rust
fn add_one(x: i32) -> i32 { x + 1 }

fn do_twice(f: fn(i32) -> i32, arg: i32) -> i32 {
    f(f(arg))
}

let result = do_twice(add_one, 5);  // 7
```

## Retornando Closures

```rust
fn returns_closure() -> impl Fn(i32) -> i32 {
    |x| x + 1
}

fn make_adder(y: i32) -> impl Fn(i32) -> i32 {
    move |x| x + y
}
```

> **Nota:** Usa `impl Fn/Trait` o `Box<dyn Fn>` para retornar closures.
