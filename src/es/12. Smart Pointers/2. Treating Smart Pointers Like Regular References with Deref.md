# Trait `Deref`

Hace que los punteros inteligentes funcionen como referencias normales con el operador `*`.

```rust
let x = 5;
let y = Box::new(x);
assert_eq!(5, *y);   // *y funciona porque Box implementa Deref
```

## Coerción Deref

Rust convierte automáticamente `&Box<T>` a `&T` (y más allá).

```rust
fn hello(name: &str) { println!("{name}"); }

let m = Box::new(String::from("Rust"));
hello(&m);  // &Box<String> -> &String -> &str (automático)
```

> **Punto clave:** La coerción Deref hace que los punteros inteligentes sean ergonómicos de usar.
