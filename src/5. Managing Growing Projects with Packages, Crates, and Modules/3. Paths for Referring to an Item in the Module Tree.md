# Paths for Referring to an Item in the Module Tree
A **path** is the way you refer to an item in the module tree.
There are two types of paths.

- An **absolute path** stars from the crate root (crate::).
- A **relative path** starts from the current module (super:: or self::)
- Paths are made of one or more identifiers separated by double colons (::).


    // In src/main.rs
    fn main() {
        // Absolute path, starting from the crate root
        let plant = my_project::garden::vegetables::Asparagus {
        name: String::from("Spargel"),
        };
    
        // Note: A relative path would be used within the same crate,
        // e.g., if you were in a different module within `my_project`.
    }
