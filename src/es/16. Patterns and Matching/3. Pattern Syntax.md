# Sintaxis de Patrones

```rust
let x = 5;

match x {
    1 => println!("one"),
    2 | 3 => println!("two or three"),       // | (or)
    4..=9 => println!("range"),              // range
    _ => println!("anything"),               // wildcard
}

// Destructuring
let (a, b, _) = (1, 2, 3);                  // tuple
let Point { x, y } = point;                  // struct

// @ bind (match + bind)
match age {
    age @ 18..=64 => println!("adult: {age}"),
    _ => (),
}

// Guards
match n {
    x if x > 0 => println!("positive: {x}"),
    _ => println!("non-positive"),
}
```

> **Punto clave:** `|` para alternativas, `..=` para rangos, `@` para vincular + coincidir, guardas para condiciones adicionales.
