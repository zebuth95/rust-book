# `cargo install`

Instala crates binarios desde crates.io en `~/.cargo/bin/`.

```bash
cargo install ripgrep     # rg
cargo install bat          # cat con resaltado de sintaxis
cargo install fd-find      # fd
```

> **Nota:** Solo funciona para crates que producen binarios (tienen un `[[bin]]` o `src/main.rs`).
