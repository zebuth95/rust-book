# OOP Characteristics in Rust

## Encapsulation

```rust
pub struct AveragedCollection {
    list: Vec<i32>,
    average: f64,           // private field
}

impl AveragedCollection {
    pub fn add(&mut self, value: i32) {
        self.list.push(value);
        self.update_average();
    }

    pub fn average(&self) -> f64 {
        self.average
    }

    fn update_average(&mut self) { /* private method */ }
}
```

## Polymorphism via Traits

```rust
pub trait Draw {
    fn draw(&self);
}

// Generics (static dispatch)
fn draw_once<T: Draw>(item: T) { item.draw(); }

// Trait objects (dynamic dispatch)
fn draw_ref(item: &dyn Draw) { item.draw(); }
```

> **Note:** Rust favors composition over inheritance. Use traits to share behavior.
