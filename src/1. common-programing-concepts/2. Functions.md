# Functions
Functions are prevalent in the Rust code. Rust uses snake case as the conventional style for functions and variable names, in which all letter are lowercase and underscores separate words.

    fn main() {
        println!("Hello, world!");
    
        another_function();
    }
    
    fn another_function() {
        println!("Another function.");
    }

We define a function by entering **fn**.

## Parameters
We can define functions to have a parameter, which are special variable that are part of a function.

    fn main() {
        another_function(5);
    }
    
    fn another_function(x: i32) {
        println!("The value of x is: {x}");
    }

## Statements and Expressions
Function bodies are made uo of a series of statements optionally ending ina expression.

- Statements are instructions that perform some action and do not return a value.
- Expressions evaluate to a resultant value.


    fn main() {
        // Statements do not return values. 'let'
        let x = plus_one(5);
    
        println!("The value of x is: {x}");
    }
    
    fn plus_one(x: i32) -> i32 {
        x + 1
    }
