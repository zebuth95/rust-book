# Cómo Escribir Pruebas

## Macros de aserción

```rust
#[test]
fn check_addition() {
    assert!(2 + 2 == 4);
    assert_eq!(2 + 2, 4);
    assert_ne!(2 + 2, 5);
}
```

## Mensajes de fallo personalizados

```rust
assert_eq!(2 + 2, 4, "2 + 2 should be 4");
```

## Esperando panics

```rust
#[test]
#[should_panic(expected = "out of bounds")]
fn index_out_of_bounds() {
    let v = vec![1, 2];
    v[10];
}
```

## Usando `Result<T, E>` en pruebas

```rust
#[test]
fn it_works() -> Result<(), String> {
    if 2 + 2 == 4 { Ok(()) } else { Err("nope".into()) }
}
```

> **Nota:** `#[should_panic]` verifica que el código entre en pánico con un mensaje coincidente.
