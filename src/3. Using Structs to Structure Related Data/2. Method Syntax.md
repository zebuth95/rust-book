# Method Syntax

Methods are functions defined **inside an `impl` block**, associated with a type.

```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }

    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width > other.width && self.height > other.height
    }
}

fn main() {
    let r = Rectangle { width: 30, height: 50 };
    println!("Area: {}", r.area());
}
```

## Associated Functions (no `self`)

Used as constructors. Called with `::`.

```rust
impl Rectangle {
    fn square(size: u32) -> Self {
        Self { width: size, height: size }
    }
}

let sq = Rectangle::square(10);
```

> **Note:** `Self` = the type after `impl`. A struct can have multiple `impl` blocks.
