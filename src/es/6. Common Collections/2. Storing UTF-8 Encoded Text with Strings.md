# Cadenas de Texto (Strings)

Rust tiene dos tipos de cadenas:

- `&str` — porción de cadena inmutable (vista a datos)
- `String` — texto UTF-8 redimensionable, asignado en el heap

```rust
let mut s = String::new();
let s = "hello".to_string();
let s = String::from("hello");
```

## Actualizando

```rust
let mut s = String::from("foo");
s.push_str("bar");         // "foobar"
s.push('!');               // "foobar!"

let s1 = String::from("Hello, ");
let s2 = String::from("world!");
let s3 = s1 + &s2;         // nota: s1 es MOVIDO aquí

let s = format!("{s2}-{s3}");  // no toma posesión
```

## ¡Sin indexación!

Las cadenas de Rust no soportan `s[0]` — los caracteres UTF-8 pueden ocupar múltiples bytes.

```rust
for c in "Зд".chars() { println!("{c}"); }   // З, д
for b in "Зд".bytes() { println!("{b}"); }   // 208, 151, 208, 180
```

> **Nota:** Usa `chars()` o `bytes()` para iterar, nunca indexación directa.
