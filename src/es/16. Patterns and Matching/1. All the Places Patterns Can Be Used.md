# Dónde se Usan los Patrones

```rust
// match arms
match value {
    Some(x) => println!("{x}"),
    None => (),
}

// if let / while let
if let Some(x) = value { }
while let Ok(val) = rx.recv() { }

// let statements
let (x, y) = (1, 2);
let Point { x, y } = point;

// for loops
for (i, val) in v.iter().enumerate() { }

// function parameters
fn foo(&(x, y): &(i32, i32)) { }
```

> **Nota:** Los patrones hacen que el acceso a datos en Rust sea seguro y expresivo.
