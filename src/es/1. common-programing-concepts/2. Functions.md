# Funciones

Definidas con `fn`. Convención: `snake_case`.

```rust
fn main() {
    greet("World");
}

fn greet(name: &str) {
    println!("Hello, {name}!");
}
```

## Parámetros y Valores de Retorno

```rust
fn add(x: i32, y: i32) -> i32 {
    x + y   // no semicolon = return expression
}
```

## Sentencias vs Expresiones

- **Sentencias** — realizan una acción, no retornan nada (`let x = 5;`)
- **Expresiones** — evalúan a un valor (`x + 1`, llamadas a funciones)

```rust
fn main() {
    let y = {
        let x = 3;
        x + 1    // expression, returns 4
    };
    println!("{y}");  // 4
}
```

> **Punto clave:** La última expresión en el cuerpo de una función es el valor de retorno (no se necesita `return`).
