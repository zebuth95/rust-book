# `Box<T>` — Asignación en el Heap

`Box<T>` almacena datos en el heap.

```rust
let b = Box::new(5);    // 5 en el heap, b en la pila
println!("{b}");         // 5
```

## Casos de uso

- Tipos recursivos (tamaño desconocido en tiempo de compilación).
- Datos grandes — transferir propiedad sin copiar.

```rust
enum List {
    Cons(i32, Box<List>),  // Box da un tamaño fijo
    Nil,
}
```

> **Nota:** `Box<T>` se libera automáticamente cuando sale de ámbito.
