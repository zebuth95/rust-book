# `RefCell<T>` — Interior Mutability

Allows mutation through an immutable reference. Borrow rules checked at **runtime** (not compile time).

```rust
use std::cell::RefCell;

let x = RefCell::new(String::from("Hello"));
let mut s = x.borrow_mut();   // mutable borrow
s.push_str(" World");
// s released here
println!("{}", x.borrow());    // "Hello World"
```

## Rules

- `borrow()` → multiple immutable borrows allowed.
- `borrow_mut()` → only one mutable borrow at a time (panics if violated at runtime).

## Common pattern: `Rc<RefCell<T>>`

Share and mutate data in single-threaded code.

```rust
let data = Rc::new(RefCell::new(5));
*data.borrow_mut() += 1;
```

> **Key point:** Use `RefCell<T>` when the compiler can't verify safety but you know it's safe.
