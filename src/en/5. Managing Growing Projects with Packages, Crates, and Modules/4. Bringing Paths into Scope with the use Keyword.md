# Bringing Paths into Scope with the use Keyword
The **use** keyword brings a path into the current scope, allowing you to refer to the item by its name without typing the full path every time.

### Example
The following code demonstrates using use to simplify the path to Asparagus.

    // In src/main.rs
    use my_project::garden::vegetables::Asparagus;
    
    fn main() {
        let plant = Asparagus {
            name: String::from("Spargel"),
        };
    }

You can use nested paths with curly brackets {} to bring multiple items from the same path into scope,
**like use std::cmp::{self, Ordering};**.

You can also use the **as** keyword to create an alias, which is useful to avoid name conflicts.
For example: **use std::fmt::Result as FmtResult;**
