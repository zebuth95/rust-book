# Workspaces

A **workspace** manages multiple crates in one project. Single `Cargo.lock`, shared output.

```
my_workspace/
├── Cargo.toml          # [workspace] members = ["adder", "add_one"]
├── adder/
│   └── src/main.rs
└── add_one/
    └── src/lib.rs
```

```toml
# Root Cargo.toml
[workspace]
members = ["adder", "add_one"]
```

```bash
cargo build   # builds all workspace members
```

> **Note:** Use workspaces for large projects composed of multiple crates.
