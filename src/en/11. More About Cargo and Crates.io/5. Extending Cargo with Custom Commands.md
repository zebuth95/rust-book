# Custom Cargo Commands

Any binary named `cargo-something` in `PATH` can be run as `cargo something`.

```bash
# Install a custom command
cargo install cargo-watch

# Run it
cargo watch
```

Popular examples: `cargo clippy` (linter), `cargo audit` (security checker).

> **Note:** This is how Cargo's extensibility system works — it's trivial to add custom subcommands.
