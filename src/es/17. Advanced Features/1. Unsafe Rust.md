# Unsafe Rust

Opcionalmente desactiva las comprobaciones de seguridad cuando necesitas control de bajo nivel.

## Cuatro superpoderes

```rust
// 1. Dereference raw pointers
let mut num = 5;
let r1 = &num as *const i32;
let r2 = &mut num as *mut i32;
unsafe { println!("{}", *r1); }

// 2. Call unsafe functions
unsafe fn dangerous() {}
unsafe { dangerous(); }

// 3. Access mutable static variables
static mut COUNTER: u32 = 0;
unsafe { COUNTER += 1; }

// 4. Implement unsafe traits
unsafe trait Foo {}
unsafe impl Foo for i32 {}
```

## Abstracciones seguras

Envuelve código unsafe en funciones seguras.

```rust
fn split_at_mut(slice: &mut [i32], mid: usize) -> (&mut [i32], &mut [i32]) {
    let len = slice.len();
    let ptr = slice.as_mut_ptr();
    assert!(mid <= len);
    unsafe {
        (
            std::slice::from_raw_parts_mut(ptr, mid),
            std::slice::from_raw_parts_mut(ptr.add(mid), len - mid),
        )
    }
}
```

> **Advertencia:** Unsafe no significa que el código sea peligroso — significa que **tú** garantizas la seguridad.
