# Closures

Closures are anonymous functions that can **capture variables** from their scope.

```rust
let x = 5;
let add = |y| y + x;    // captures x immutably
println!("{}", add(3));  // 8
```

## Trait bounds

| Trait | Captures | Called |
|---|---|---|
| `Fn` | Immutable borrow (`&T`) | Many times |
| `FnMut` | Mutable borrow (`&mut T`) | Many times |
| `FnOnce` | Move/consume (`T`) | Once |

## `move` keyword

Forces the closure to take ownership of captured variables (required when spawning threads).

```rust
let data = vec![1, 2, 3];
let closure = move || {
    println!("{:?}", data);  // data moved here
};
closure();
// println!("{:?}", data);  // ❌ data moved
```

> **Key point:** Closures infer their trait based on what they do with captured variables.
